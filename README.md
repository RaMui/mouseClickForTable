MouseClickForTable是一个基于jquery编写的前端插件，旨在为html网页的table提供更多功能。

___

# 功能说明

1. 鼠标左键点击表头进行排序,可以正/倒序排序。
>![](\img\01.gif)
2. 鼠标右键点击表头进行筛选,可以多选。
>![](\img\02.gif)
3. 搜索框可以查找需要筛选的数据。
>![](\img\03.gif)
4. 使用该插件你的table需要按以下格式来写:
```
<table id="tableId">
    <thead>
        <tr>
            <th></th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td></td>
        </tr>
    </tbody>
</table>
```
点击[试用](https://ramui.github.io/mouseClickForTable/)来自己尝试一下吧。
